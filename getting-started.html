Getting Started

Learn how to set up and start using Water Framework in your projects

Requirements

Java 17+

Make sure you have Java 17 or later installed

Node.js 18.20.8+

Required for the Water generator

Gradle

Build tool for Java projects

Yeoman

Required for generating Water Framework projects

Installation

1. Install Node.js

If you're using NVM, switch to the required version:

nvm use 18.20.8

2. Install Water Generator

Install the Water generator globally:

npm install -g yo water-generator

3. Verify Installation

Check available commands:

yo water:help

Quick Start

Create a New Project

# Create a new project
yo water:new my-project

# Navigate to project directory
cd my-project

# Build the project
yo water:build-all
