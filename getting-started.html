Getting Started with Water Framework

Introduction

Welcome to Water Framework! This guide will help you get started with our framework and show you how to build your first application.

Prerequisites

  • Java 17 or higher
  • Maven 3.6 or higher
  • Your favorite IDE (IntelliJ IDEA, Eclipse, or VS Code)

Installation

To get started, you'll need to add the Water Framework dependencies to your project.

Maven


<dependency>
    <groupId>org.waterframework</groupId>
    <artifactId>water-core</artifactId>
    <version>1.0.0</version>
</dependency>

Gradle


implementation 'org.waterframework:water-core:1.0.0'

Quick Start

Here's a simple example to get you started:


import org.waterframework.core.WaterApplication;
import org.waterframework.core.annotation.WaterComponent;

@WaterComponent
public class MyApplication {
    public static void main(String[] args) {
        WaterApplication.run(MyApplication.class, args);
    }
}
